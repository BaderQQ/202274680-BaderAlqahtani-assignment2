# Personal Portfolio Web Application (Assignment 2)

## 1. Project Description

This project is an improved version of my personal portfolio website developed using HTML, CSS, and JavaScript.

The website includes the following sections:

- **About Me** – A personal introduction and background.
- **Projects** – Displays my projects with descriptions and images.
- **Contact** – A contact form with validation and user feedback.

This version enhances the original project by adding **dynamic content, data handling, animations, and user feedback** to improve interactivity and user experience.

---

## 2. Features

### Dynamic Content
- Live project search that filters projects by title while typing.

### Data Handling
- Theme preference (dark/light mode) is saved using `localStorage`.
- Contact form includes input validation and confirmation messages.

### Animations & Transitions
- Hover effects on buttons and project cards.
- Fade-in animation when sections appear on screen.
- Smooth transitions for theme switching and UI elements.

### Error Handling & User Feedback
- Error messages for invalid form inputs.
- Success message after form submission.
- Empty state message when no projects match the search.
