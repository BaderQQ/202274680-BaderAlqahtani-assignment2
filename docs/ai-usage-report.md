# AI Usage Report

## 1. AI Tools Used

### ChatGPT
I used ChatGPT as the main AI tool during this assignment.

### GitHub Copilot
I used GitHub Copilot for code suggestions and auto-completion while coding.

---

## 2. How AI Was Used

AI tools were used to assist with the following parts of the project:

- Implementing the **project search feature** (dynamic content)
- Fixing and improving **form validation logic**
  - Correcting email validation (regex issue)
  - Adding name validation (letters only)
- Improving **user feedback messages**
- Helping structure and improve **project documentation**

---

## 3. Benefits of Using AI

- Helped me understand how to implement interactive features in JavaScript
- Made debugging faster (especially validation issues)
- Improved the overall user experience of the website
- Saved time when writing repetitive code
- Helped improve the clarity and structure of documentation

---

## 4. Challenges Faced

- Some AI-generated code contained small errors (e.g., incorrect regex syntax)
- Some solutions were more complex than required for the assignment
- Required careful review to ensure correctness

---

## 5. Learning Outcomes

By using AI tools, I learned:

- How to create dynamic features such as live search
- How to validate form inputs properly
- How to use `localStorage` to store user preferences
- The importance of reviewing and understanding AI-generated code

---

## 6. Responsible Use of AI

To ensure proper and ethical use of AI:

- I reviewed and understood all AI-generated code before using it
- I fixed errors and modified the code to match assignment requirements
- I simplified solutions when necessary
- I did not copy code without understanding it

AI was used as a **support tool to enhance learning**, not as a replacement for my own work.
